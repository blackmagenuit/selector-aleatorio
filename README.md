# 🎡 Ruleta de Actividades

Una ruleta interactiva para elegir actividades al azar. Cargás tus tareas, girás, y el azar decide por vos — con animación, sonidos chiptune estilo Nintendo, confetti y mensajes cálidos en cada tarea completada. Cuando terminás todo, Charmander baila para celebrarlo.
